# Weather App

A weather app that shows the current weather and forecast either by location or searching for a city.

## Installation

```
npm install
```

## Technologies

- HTML
- CSS
- JavaScript
- Bootstrap
- Axios

You can see this project [here](https://youthful-knuth-a37c6b.netlify.app/)
